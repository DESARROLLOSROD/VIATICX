# VIATICX Frontend

Aplicación React con TypeScript y Tailwind CSS para el sistema de gestión de gastos y viáticos.

## Stack Tecnológico

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Query (TanStack Query)
- React Router v6
- Zustand (State Management)
- React Hook Form + Zod

## Instalación

```bash
npm install
```

## Configuración

1. Copiar `.env.example` a `.env`
2. Configurar `VITE_API_URL` con la URL del backend

## Desarrollo

```bash
npm run dev
```

Aplicación disponible en: http://localhost:3000

## Build

```bash
npm run build
npm run preview
```

## Testing

```bash
npm run test
npm run test:coverage
```
